AI Video Sort is an intelligent media-management tool that automatically analyzes, sorts, and organizes videos/images using AI-powered classification.  
This project focuses on building a clean UI, fast processing, and accurate media categorization based on content.


## Features

-  AI-based video/image classification
-  Automatic folder sorting by labels
-  Fast processing using optimized pipeline
-  Model-based context understanding
- Clean and responsive UI
- Progress tracking while media is being processed
- API-driven architecture (Frontend + FastAPI backend)
